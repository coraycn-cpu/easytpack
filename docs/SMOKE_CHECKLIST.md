# EasytPack 冒烟清单（手工）

每次发版或大改 Studio / 导出前，按下列路径点一遍。本地无账号即可。

## 1. 新建与进入画布

- [ ] 首页上传图 + 样衣码 → **进入画布**
- [ ] 识别款式类型后，可「暂不生成」直接用原图，或生成平铺正面
- [ ] 存储满时有明确报错，并链到「我的项目」

## 2. 画布标注

- [ ] 方框 / 画笔 / 箭头 / 尺寸线可画可选
- [ ] 工艺行 ↔ 画布标注可互跳高亮
- [ ] 尺寸线可关联尺码表部位

## 3. AI

- [ ] **AI 一键标注**：问答弹窗（≤5 题）后写入工艺/物料/尺寸初稿
- [ ] **一键补全**：只补缺项，不覆盖已有有效内容
- [ ] 生成背面 / 线稿：失败时**不**写入假图；失败角标仅历史占位图出现
- [ ] 失败画板可「修正」重试或删除

## 4. 合规与导出

- [ ] 侧栏合规项可点击跳到对应 Tab / 画布提示
- [ ] 导出页：PDF（打印 A4 横向）/ Excel / 合拼大图
- [ ] 打印预览含封面与标注；Excel 含视图 Sheet

## 5. 本地存储

- [ ] 「我的项目」显示用量拆分
- [ ] 清理缓存不删项目
- [ ] 可导出项目 JSON 备份；可导出质量日志 JSONL

## 6. 回归注意

- [ ] 刷新后项目仍在（同浏览器）
- [ ] 复制 / 删除项目正常
