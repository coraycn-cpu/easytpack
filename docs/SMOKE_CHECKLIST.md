# EasytPack 冒烟清单（手工）

每次发版或大改 Studio / 导出 / 云同步前，按下列路径点一遍。

## 1. 新建与进入画布

- [ ] 首页为空白画布壳（不自动打开旧款），有「怎么开始」说明
- [ ] 未登录可见登录 / 注册；登录后可见最近项目
- [ ] 点「新建款式」上传图 + 样衣码 → 进入画布
- [ ] 识别款式类型后，可「暂不生成」直接用原图，或生成平铺正面
- [ ] 存储满时有明确报错，并链到「我的项目」

## 2. 画布标注

- [ ] 方框 / 画笔 / 箭头 / 尺寸线可画可选
- [ ] 工艺行 ↔ 画布标注可互跳高亮
- [ ] 尺寸线可关联尺码表部位
- [ ] 顶栏「我的项目」下拉不被工具栏挡住

## 3. AI

- [ ] **AI 一键标注**：问答弹窗（≤5 题）后写入工艺/物料/尺寸初稿
- [ ] **一键补全**：只补缺项，不覆盖已有有效内容
- [ ] 生成背面 / 线稿：失败时**不**写入假图；失败角标仅历史占位图出现
- [ ] 失败画板可「修正」重试或删除

## 4. 合规与导出

- [ ] 侧栏合规项可点击跳到对应 Tab / 画布提示
- [ ] 导出页：PDF（打印 A4 横向）/ Excel / 合拼大图
- [ ] 打印预览含封面与标注；Excel 含视图 Sheet

## 5. 本地存储

- [ ] 「我的项目」显示用量拆分
- [ ] 清理缓存不删项目
- [ ] 可导出项目 JSON 备份；可导出质量日志 JSONL

## 6. 回归注意

- [ ] 刷新后项目仍在（同浏览器）
- [ ] 复制 / 删除项目正常

## 7. 云端账号与跨设备（`feat/phase2-cloud` Preview）

前置：已按 `docs/SUPABASE_SETUP.md` 配好钥匙、跑过 `schema.sql`、Storage 有 `style-images`。

- [ ] 注册 / 登录成功；未登录仍可本机做款
- [ ] 登录后自动或手动「双向同步」不报错
- [ ] 「我的项目」可切换自动/手动同步；手动时保存不自动上传，点同步才上传
- [ ] 打开 `/account`：能看邮箱、同步偏好、本月 AI 用量、邀请好友入口；团队升级显示即将开放
- [ ] 登录后调用一次 AI → Supabase `ai_usage` 有新行；额度用尽时返回提示
- [ ] 勾选质量池 consent 后的视角生图，尽力写入 `ai_events`（失败不影响本机）
- [ ] 用户中心复制邀请链接 → 另一邮箱注册 → 邀请人积分 +50，成功数最多计到 5
- [ ] AI 额度上限 = 免费档 + 邀请积分（用户中心用量行可核对）
- [ ] Supabase **tech_packs** 出现对应行
- [ ] **style-images** 桶内出现 `{userId}/{projectId}/…` 文件
- [ ] 换浏览器 / 无痕：登录同一账号 → 拉取后能打开旧款且**图片正常**
- [ ] 断网时仍可本机编辑；恢复后点同步，失败时有黄字提示且本机不丢稿
- [ ] 删除项目后云端行与图片尽量清理干净
